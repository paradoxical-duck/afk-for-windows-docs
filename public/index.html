Install

Install with speech model Installer only
  1. For a first-time setup, download AFK with speech model.
  2. Extract the ZIP into a normal folder. Do not run the installer from inside the ZIP preview.
  3. Double-click AFK Setup 1.0.0.exe. The setup wizard installs AFK and copies the bundled Parakeet speech model automatically.
  4. If Parakeet is already installed, use Installer only and double-click AFK Setup 1.0.0.exe.
  5. Open AFK, choose your microphone, and leave it in the system tray.
  6. Turn on Start on login if you want the hotkeys ready after every reboot.

First Run Checklist

1Pick the correct Windows microphone in AFK Settings.
2Turn on Start on login so hotkeys work after a reboot.
3Install both Clarify models if you want Ctrl+Alt+K correction.
4Quit from the tray only when you want all global hotkeys to stop.

Official Models

AFK uses ONNX Parakeet for local speech recognition and GGUF Gemma models for Clarify. The installer includes the runtime; the large model files live in your roaming AFK model folder.

$models = "$env:APPDATA\AFK\models\clarify"
New-Item -ItemType Directory -Force -Path $models

$py = "$env:LOCALAPPDATA\Programs\AFK\resources\python\runtime\python.exe"
@"
from huggingface_hub import snapshot_download
snapshot_download(repo_id="ggml-org/gemma-3-270m-GGUF", allow_patterns="gemma-3-270m-Q8_0.gguf", local_dir=r"$models")
snapshot_download(repo_id="ggml-org/gemma-4-E2B-it-GGUF", allow_patterns="gemma-4-E2B-it-Q8_0.gguf", local_dir=r"$models")
"@ | & $py -

Training

Training lets AFK remember how you personally say words that Parakeet gets wrong. Type the intended word or phrase, record yourself saying it, and AFK stores a local correction that is applied after transcription.

Word trainingBest for names, usernames, acronyms, and common phrases that are repeatedly misheard.
Private storageTraining data stays in your AFK app data folder on Windows and can be deleted item by item from the Train tab.
Exact outputThe phrase AFK inserts keeps the capitalization and symbols you typed in the training output.

Triggers

Triggers turn a spoken phrase into reliable reusable text. Trigger matching is case-insensitive, so AFK treats “my email is” and “My Email Is” as the same trigger while preserving the exact output you typed.

Autofill Keep the phrase, add the output

Say my github username is and AFK outputs my github username is paradoxical_duck.

Autoreplace Replace the phrase entirely

Say github username and AFK outputs paradoxical_duck.

Paste Behavior

After transcription, AFK types into the active text field without replacing your clipboard. It only copies the transcript to the clipboard if keyboard insertion fails.

Troubleshooting

No speech detected

Check Windows input volume, confirm the microphone is not muted, then pick the correct microphone in AFK settings.

Clarify unavailable

Install both GGUF model files into %APPDATA%\AFK\models\clarify. Restart AFK from the tray after downloading.

Weird tokens appear in transcripts

Install the current fixed build from this page. It strips chat-template tokens such as <|im_start|> before text reaches your app or clipboard.

Hotkeys stop working

AFK must be running in the tray for global hotkeys. Enable Start on login in Settings.
