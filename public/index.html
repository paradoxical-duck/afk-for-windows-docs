Install

  1. Download AFK Setup 0.1.0 ZIP.
  2. Extract the ZIP, then run AFK Setup 0.1.0 FIXED.exe.
  3. Windows SmartScreen may ask you to confirm because this local build is unsigned.
  4. Open AFK once, choose your microphone, and leave it in the system tray.
cd "$env:LOCALAPPDATA\Programs\AFK"
.\AFK.exe

First Run Checklist

1Pick the correct Windows microphone in AFK Settings.
2Turn on Start on login so hotkeys work after a reboot.
3Install both Clarify models if you want Ctrl+Alt+K correction.
4Quit from the tray only when you want all global hotkeys to stop.

Official Models

AFK uses ONNX Parakeet for local speech recognition and GGUF Gemma models for Clarify. The installer includes the runtime; the large model files live in your roaming AFK model folder.

$models = "$env:APPDATA\AFK\models\clarify"
New-Item -ItemType Directory -Force -Path $models

$hf = "$env:LOCALAPPDATA\Programs\AFK\resources\python\.venv\Scripts\hf.exe"
& $hf download ggml-org/gemma-3-270m-GGUF --include gemma-3-270m-Q8_0.gguf --local-dir $models
& $hf download ggml-org/gemma-4-E2B-it-GGUF --include gemma-4-E2B-it-Q8_0.gguf --local-dir $models

Hotkeys

CtrlSpaceHold to dictate
CtrlShiftSpaceToggle dictation
CtrlAltKClarify selected text

AFK keeps running from the system tray. Closing the window hides it; quitting from the tray fully stops hotkeys.

Paste Behavior

After transcription, AFK pastes into the active text field and restores your previous clipboard. If there is no active text caret, it copies the transcript to the clipboard instead.

Troubleshooting

No speech detected

Check Windows input volume, confirm the microphone is not muted, then pick the correct microphone in AFK settings.

Clarify unavailable

Install both GGUF model files into %APPDATA%\AFK\models\clarify. Restart AFK from the tray after downloading.

Weird tokens appear in transcripts

Install the current fixed build from this page. It strips chat-template tokens such as <|im_start|> before text reaches your app or clipboard.

Hotkeys stop working

AFK must be running in the tray for global hotkeys. Enable Start on login in Settings.
